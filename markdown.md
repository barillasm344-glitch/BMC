## 📖 Descripción

**BMC VAPING** es una tienda en línea especializada en productos de vapeo de alta calidad. Este proyecto es un sitio web responsive, moderno y funcional que permite a los usuarios explorar productos, filtrar por categorías, agregar al carrito y contactar con el equipo de ventas.

### 🎯 Características Principales

- ✅ **Verificación de Edad** (18+) con persistencia en localStorage
- ✅ **Catálogo de Productos** con filtros por categoría
- ✅ **Carrito de Compras** funcional con localStorage
- ✅ **Búsqueda en tiempo real** de productos
- ✅ **Diseño Mobile-First** completamente responsive
- ✅ **Menú Hamburguesa** para dispositivos móviles
- ✅ **Formulario de Contacto** con validación
- ✅ **FAQ Acordeón** interactivo
- ✅ **Sección de Testimonios**
- ✅ **Optimización SEO** (Meta tags, Open Graph)
- ✅ **Accesibilidad** (ARIA labels, roles semánticos)

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos modernos con variables CSS
- **JavaScript (ES6+)** - Interactividad y lógica de negocio
- **Font Awesome 6** - Iconos profesionales
- **Google Fonts (Inter)** - Tipografía moderna
- **LocalStorage** - Persistencia de datos (carrito, edad)
